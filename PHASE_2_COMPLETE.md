1. **Accordion.tsx** - Reusable accordion component
   - Single/multiple open items
   - Animated chevron rotation
   - Hover effects

2. **HowItWorks.tsx** - 4-step process section
   - Desktop: horizontal timeline
   - Mobile: vertical timeline with connector lines
   - Animated step numbers
   - Bottom stats row

3. **Stats.tsx** - Animated metrics grid
   - 6 real-time counters
   - Animated number counting
   - Progress bars
   - Live indicator badge

4. **Testimonials.tsx** - Social proof carousel
   - Desktop: 3-column grid
   - Mobile: Full carousel with navigation
   - Star ratings (5 stars each)
   - Avatar + author info
   - Smooth transitions

5. **Pricing.tsx** - 3-tier pricing table
   - Free, Pro, Enterprise tiers
   - Pro highlighted as "Most Popular"
   - Feature checklist with icons
   - CTA buttons per plan
   - FAQ section

6. **FAQ.tsx** - Accordion FAQ section
   - 9 comprehensive questions
   - Accordion component
   - Contact support CTA
   - Fully responsive

7. **CTA.tsx** - Final conversion section
   - Large headline + subheading
   - 3 CTA buttons (Telegram, Dashboard, Docs)
   - Trust indicators
   - Stats preview

8. **pages/index.tsx** - Main landing page
   - Combines all 8 sections + Header/Footer
   - Complete page flow
   - Ready to deploy
